Screen-Hierarchie:
Tab: Home
  ├── Tool-Liste (Feed)
  ├── Tool-Detail → Buchung erstellen
  └── Bewertung abgeben
Tab: Suche
  ├── Suche + Filter
  └── Kategorien-Übersicht
Tab: Anbieten (+)
  ├── Werkzeug erstellen
  └── Foto aufnehmen (Kamera)
Tab: Profil
  ├── Mein Profil
  ├── Meine Werkzeuge
  └── Meine Buchungen
Drawer (☰):
  ├── Benachrichtigungen
  ├── Einstellungen
  ├── Hilfe / FAQ
  └── Abmelden
Backend-Änderung (minimal):
1. Auth-Middleware: Bearer Token als Alternative zu Cookie
2. Neuer Endpoint: POST /api/v1/notifications/push-token
3. Push-Notification Service (Expo Push API)

Projekt-Struktur (neuer Ordner: mobile/)

├── app/ # Expo Router (file-based routing)

│ │ ├── index.tsx # Home (Tool-Feed)
│ │ ├── search.tsx # Suche
│ │ ├── create.tsx # Werkzeug anbieten
│ │ └── profile.tsx # Profil
│ ├── tool/[id].tsx # Tool-Detail
│ ├── booking/[id].tsx # Buchungs-Detail
│ ├── (auth)/ # Auth-Screens
│ │ ├── login.tsx
│ │ └── register.tsx
│ └── _layout.tsx # Root Layout (Drawer)
├── components/ # Wiederverwendbare Komponenten
├── hooks/ # Custom Hooks (useAuth, useTools...)
├── lib/ # API Client, Query Keys, Utils
├── types/ # TypeScript Typen (shared mit Web?)
